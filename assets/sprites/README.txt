PHYSICAL: TECH - Player Sprites
================================

Place your 144 player pixel art sprites in this folder.

File naming convention (must match players.csv):
- batbold.png
- dolgorsuren.png
- ganzorig.png
- ... (144 files total)

Recommended specs:
- Size: 64x64 or 128x128 pixels
- Format: PNG with transparency
- Style: Pixel art / full body

The sprites will appear in:
- Player selection screen (arcade style)
- Match preview screen
- In-game character display
- Victory celebration screen

If a sprite is missing, the system will show a default emoji fallback.
