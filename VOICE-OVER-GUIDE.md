## ‚öôÔ∏è Technical Specifications

### Audio Format
- **Format:** MP3 (recommended) or WAV
- **Sample Rate:** 44.1 kHz or 48 kHz
- **Bit Rate:** 192 kbps for voice, 128-192 kbps for music
- **Channels:** Mono for voice (smaller file), Stereo for music

### Voice Characteristics
- **Tone:** Low, masculine, authoritative
- **Style:** Sports announcer / game show host energy
- **Clarity:** Crystal clear, no background noise
- **Volume:** Normalized to -3dB to -6dB (loud but not clipping)

### File Size Guidelines
- **Countdown:** < 500 KB
- **Game Intros:** < 1 MB each
- **Victory Announcements:** < 300 KB each
- **Championship Announcements:** < 500 KB each

## üé§ Recording Tips

### DIY Recording (Budget Option)
1. Use a quiet room with soft furnishings (reduces echo)
2. Record on phone or laptop microphone
3. Free software: Audacity (Windows/Mac/Linux)
4. Stand 6-12 inches from microphone
5. Use noise reduction in Audacity if needed

### Text-to-Speech (AI Option)
**Free AI Voice Generators:**
- **ElevenLabs** (https://elevenlabs.io) - Best quality, 10 mins free/month
- **Play.ht** (https://play.ht) - Good quality, free tier available
- **Uberduck** (https://uberduck.ai) - Free, multiple voices

**Recommended AI Voice Settings:**
- Select "Deep Male" or "Announcer" voice
- Stability: 75-85%
- Clarity: 80-90%
- Style exaggeration: 50-70% (for excitement)

### Professional Option
- Fiverr: $5-20 per file
- Upwork: $20-50 for all files
- Voice acting subreddits: Sometimes free for portfolio

## üîß How the System Works

### Automatic Fade In/Out
The audio manager automatically handles:
- **Fade out** background music before voice-overs (1-2 seconds)
- **Fade in** background music after voice-overs (0.5-1 seconds)
- **Crossfading** between different music tracks (2 seconds)

### Fallback System
If an audio file is missing:
- Game intro missing ‚Üí Background music starts immediately
- Victory announcement missing ‚Üí Plays procedural fanfare sound
- Championship missing ‚Üí Plays procedural fanfare + starts celebration music

**This means the tournament works even without all files!**

## üéØ File Naming is CRITICAL

The system expects **exact file names**. Even one wrong character will cause the file not to play.

### ‚úÖ Correct:
- `victory-team-1-2.mp3`
- `championship-team-10.mp3`
- `game-intro-3.mp3`

### ‚ùå Wrong:
- `victory-team-1&2.mp3` (use hyphen, not &)
- `championship-team-10 .mp3` (space before extension)
- `Game-Intro-3.mp3` (capital letters)
- `game-intro-3.MP3` (wrong case)

**Use exactly the file names listed in this guide!**

## üìû Need Help?

**File not playing?**
1. Check file name spelling (case-sensitive!)
2. Check file is in `assets/audio/` folder
3. Check browser console for errors
4. Try refreshing page (Ctrl+F5)

**Audio too quiet/loud?**
- Adjust SFX volume slider in game
- Or normalize audio file to -3dB to -6dB

**Timing feels off?**
- Adjust fade durations in `audio-manager.js`
- Look for fadeOutMusic() and fadeInMusic() calls
